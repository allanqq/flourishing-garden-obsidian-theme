# Flourishing Garden

A calm Obsidian theme for reading, writing, research, and daily notes.

It has a soft garden feel without being busy: warm light mode, quiet dark mode, readable serif notes, gentle green accents, clean code blocks, and simple rounded details.

## Install

1. Download the latest release.
2. Put `manifest.json`, `theme.css`, and `screenshot.png` in:

```txt
.obsidian/themes/Flourishing Garden/
```

3. In Obsidian, go to **Settings -> Appearance -> Themes**.
4. Choose **Flourishing Garden**.

For extra options, use the Style Settings plugin.
