# Flourishing Garden

Flourishing Garden is a bio-inspired Obsidian theme for deep work, research, journaling, coding, and creative knowledge gardens. It uses calm botanical palettes, soft organic geometry, growth-ring heading hierarchy, specimen-card surfaces, and subtle mycelial interaction cues without leaning on decorative nature cliches.

## Palettes

- **Morning Canopy**: warm light mode with soil, moss, sage, pollen, clay, and water tones.
- **Mycelial Night**: low-glare dark mode with lichen text, moss accents, amber glow, teal links, coral warnings, and spore purple highlights.

## Features

- Serif-forward reading and editing typography.
- Rounded, quiet panels and controls.
- Growth-ring H1/H2 underlines.
- Seed-like tags and softly colored field-note callouts.
- Compact earthy code blocks.
- Field-notebook texture and specimen-style metadata surfaces.
- Calm graph and canvas styling inspired by mycelial networks and biological cells.
- Style Settings support for accent mode, density, typography mode, heading ornament, and code block style.

## Installation

1. Create a folder named `Flourishing Garden` in your vault's `.obsidian/themes/` directory.
2. Copy `manifest.json`, `theme.css`, and `screenshot.png` into that folder.
3. In Obsidian, open **Settings -> Appearance -> Themes**.
4. Select **Flourishing Garden**.

## Community Theme Files

This repository follows the standard Obsidian community theme layout:

- `manifest.json`
- `theme.css`
- `screenshot.png`
- `versions.json`

For tuning options, install the community **Style Settings** plugin and adjust the Flourishing Garden section.
