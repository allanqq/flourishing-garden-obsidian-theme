# Eudaimonia Grove

Eudaimonia Grove is a bio-inspired Obsidian theme for deep work, research, journaling, coding, and creative knowledge gardens. It uses calm botanical palettes, soft organic geometry, growth-ring heading hierarchy, and subtle mycelial interaction cues without leaning on decorative nature cliches.

## Palettes

- **Morning Canopy**: warm light mode with soil, moss, sage, pollen, clay, and water tones.
- **Mycelial Night**: low-glare dark mode with lichen text, moss accents, amber glow, teal links, coral warnings, and spore purple highlights.

## Features

- Serif-forward reading and editing typography.
- Rounded, quiet panels and controls.
- Growth-ring H1/H2 underlines.
- Seed-like tags and softly colored field-note callouts.
- Compact earthy code blocks.
- Calm graph and canvas styling inspired by mycelial networks and biological cells.
- Style Settings support for accent mode, density, typography mode, heading ornament, and code block style.

## Installation

1. Copy the `Eudaimonia Grove` folder into your vault's `.obsidian/themes/` directory.
2. In Obsidian, open **Settings -> Appearance -> Themes**.
3. Select **Eudaimonia Grove**.

For tuning options, install the community **Style Settings** plugin and adjust the Eudaimonia Grove section.
